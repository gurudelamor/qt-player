------------------------------------------------------------------------------------

Sinergia/
├── controllers/        # Controladores: manejan eventos y lógica de interacción
│   └── player_controller.py
│
├── services/           # Servicios: encapsulan lógica de negocio (ej. reproducción, playlist)
│   └── player_service.py
│
├── models/             # Modelos: entidades de datos (no se usan aún, preparados para futuro)
│   └── (Etc.)
│
├── views/              # Vistas: todo lo visual y estético
│   ├── icons/          # Recursos gráficos (SVG, PNG)
│   ├── skins/          # Estilos .qss para personalizar apariencia
│   └── ui/             # Archivos generados por Qt Designer (Ui_MainPlayer, layouts)
│
└── README.md           # Documentación inicial del proyecto
